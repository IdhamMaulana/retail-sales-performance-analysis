# 📊 Retail Sales Performance Analysis
## 📌 Ringkasan Proyek
Proyek ini bertujuan menganalisis performa penjualan retail untuk mengidentifikasi pendorong pendapatan (revenue drivers), kesenjangan performa kategori, serta peluang peningkatan bisnis.

Analisis dilakukan menggunakan Microsoft Excel dengan pendekatan berbasis KPI dan insight-driven decision making.
